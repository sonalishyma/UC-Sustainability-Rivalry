# UC Patterns of Sustainability — interactive site (SYN 100, Team 3)

Single self-contained `index.html` — zero JS dependencies, all real data from
`Team_3_Metrics_Data_Chart.xlsx` embedded. Works opened straight from disk.

Features: standings with the UCI/UCSB tie surfaced · interactive 63-cell rank grid
(hover any cell for the real stat + measurement notes) · per-campus profiles ·
targets deep-dive (EUI/water, % vs own goal) · LEED stock by tier · Patterns to Adopt ·
methods & honesty notes with sources.

## Deploy (GitHub Pages)
Create a repo → upload `index.html` → Settings → Pages → deploy from `main`, root.
